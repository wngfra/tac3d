Working with Probes and Monitors in NxNet¶

Probes enable you to see how the network changes over time. Below you should find examples on how to create probes and helper utilities to plot probes.

# This example shows how to configure probes for a single compartment
compartment = net.createCompartment(p)
# Create a compartment probe to probe the compartment states:
# compartment current(U) and compartment voltage(V)
# probeConditions=None implies default probe conditions
# will be used for each probe
params = [ProbeParameter.COMPARTMENT_CURRENT, ProbeParameter.COMPARTMENT_VOLTAGE]
probes = compartment.probe(params, probeConditions=None)

# This example shows how to configure probes for a compartment group
compartmentGrp = net.createCompartmentGroup(size=10, prototype=p)

# Create a compartment group probe to probe the compartment states:
# compartment current(U) and compartment voltage(V)
# Since no ProbeCondition is specified, a default probe condition
# will be used.
probes = compartmentGrp.probe([ProbeParameter.COMPARTMENT_CURRENT,
                                   ProbeParameter.COMPARTMENT_VOLTAGE])

You may use any plotting libraries to plot probes. NxSDK uses matplotlib to plot probes. See tutorials for examples.

To quickly plot a list of probes, use plotProbes mentioned below.

plotutils.plotProbes()¶

Plots a list of probes.

Valid arguments are either a single list of NxProbes/ProbeSets or a nested sequence of NxProbes/ProbeSets with uniform dimensions like (4*3) signifying four compartments (groups) and three kinds of probes.

Note

To customize the axes and other features, you may modify the figure object returned from this method.

Usage

Plot list of NxProbes

probes = compartment.probe([nx.ProbeParameter.COMPARTMENT_CURRENT,
                             nx.ProbeParameter.COMPARTMENT_VOLTAGE], probeConditions=None)

# ... net.run() ...

fig = nx.plotProbes(probes)
plt.show()

Plot a nested list of NxProbes (typically from many components)

probes = []
for i in range(0, 4):
    probeList = compartment.probe([nx.ProbeParameter.COMPARTMENT_CURRENT,
                                    nx.ProbeParameter.COMPARTMENT_VOLTAGE,
                                    nx.ProbeParameter.SPIKE], probeConditions=None)
    probes.append(probeList)

# ... net.run() ...

fig = nx.plotProbes(probes)
plt.show()

Plot a nested list of ProbeSets

cg1 = net.createCompartmentGroup(size=2, prototype=prototype1)
cg2 = net.createCompartmentGroup(size=2, prototype=prototype1)

probeParameters = [nx.ProbeParameter.COMPARTMENT_CURRENT,
                    nx.ProbeParameter.COMPARTMENT_VOLTAGE,
                    nx.ProbeParameter.SPIKE]
probeConditions = None

probes1 = cg1.probe(probeParameters, probeConditions)
probes2 = cg2.probe(probeParameters, probeConditions)

# ... net.run() ...

fig = nx.plotProbes(probes = [probes1, probes2])
plt.show()
Parameters

probes -- list or list[list] of NxProbes/ProbeSets

Returns

figure

class nxsdk.graph.monitor.nx_probes.NxProbe(probeTypeEnum, nodeId, parameterToProbe, probeCondition=None)¶

NxProbe provides the user the ability to probe the dynamic state variables of the various nodeset elements - compartments, connections, etc. - defined in an NxNet. Thus a "node" could either be a compartment or a connection, etc.

property data¶

Returns the actual values of the node state that was probed in a list. Thus, for example, data[5] gives the value measured by the probe at timestep=5.

plot(**kwargs)¶

Generates a plot of the values of nodestate probed vs. the time steps at which the probe data was collected.

property timeVector¶

Returns the time series vector for the corresponding probe - i.e. the vector of time steps at which the node state was probed. For example, if a compartment state was measured after every 3 timsteps, starting at timestep =5, then timeVector = [5, 8, 11, 14,...]
