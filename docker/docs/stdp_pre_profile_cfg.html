Stdp Pre Profile Config¶

class nxsdk.arch.n2a.graph.nodesets.stdp_pre_profile_cfg_obj_inf.StdpPreProfileCfgObjInf(updateOnXspike=0, updateOnX1=0, updateOnX2=0, updateOnRewardSpike=0, updateOnReward=0, updateAlways=0, rewardAxon=0, numTraceHist=0, numTraces=0, stdpProfile=0)¶

Represents StdpPreProfileCfg NodeSet which controls how synaptic pre traces get updated and allows to assign learning rules to synapses by input axon. Each StdpPreProfileCfg entry is associated with one or more input axons.

Size: 16
