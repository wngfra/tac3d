Synapse Format¶

class nxsdk.arch.n2a.graph.nodesets.synapse_fmt_obj_inf.SynapseFmtObjInf(wgtLimitMant=0, wgtLimitExp=0, wgtExp=0, discMaxWgt=0, learningCfg=0, tagBits=0, dlyBits=0, wgtBits=0, reuseSynData=0, numSynapses=0, cIdxOffset=0, cIdxMult=0, skipBits=0, idxBits=0, synType=0, fanoutType=0, compression=0, stdpProfile=0, ignoreDly=0)¶

Represents SynapseFmt NodeSet which specifies shared synaptic format parameters. Every synapse defined in the Synapses NodeSet points to a single SynapseFmt entry which describes the precision of various synaptic state variables, compression and encoding details, which learning rule is assigned to a synapse and other shared properties.

Size: 16

Note

If synapticFmt bits are not set correctly to accommodate the synaptic values, truncation of LSB would be performed which results in limited precision. For details as how the LSB are truncated, please refer to Synapses section.

property wgtExp¶

(int4) Specifies the exponent by which the synaptic weight is scaled before accumulation in DendAccum \(accWgt=wgt*2^{ 8-wgtPrecision+wgtExp}\).

property wgtLimitExp¶

(uint4) Specifies the exponent of the maximum weight to be accumulated in DendAccum \(accWgt=sgn(wgt)*min(wgtLimit, wgt|*2^{8-wgtPrecision+wgtExp}\), where \(wgtLimit=wgtLimitMant*2^wgtLimitExp\).

property wgtLimitMant¶

(uint4) Specifies the mantissa of the maximum weight to be accumulated in DendAccum \(accWgt=sgn(wgt)*min(wgtLimit, |wgt|*2^{8-wgtPrecision+wgtExp}\), where \(wgtLimit=wgtLimitMant*2^wgtLimitExp\). Note: If wgtLimitMant=0, the weight is not capped.
