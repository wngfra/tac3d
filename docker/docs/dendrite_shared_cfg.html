Dendrite Shared Configuration¶

class nxsdk.arch.n2a.graph.nodesets.dendrite_shared_cfg_obj_inf.DendriteSharedCfgObjInf(gaussian=0, noiseExp0=0, noiseExp1=0, noiseMantOffset0=0, noiseMantOffset1=0, noiseAtDendOrVm=0, refractInhibited=0, disableInhibited=0, negVmLimit=0, posVmLimit=0, dmOffsets=0, dsOffset=0)¶

Represents DendriteSharedCfg NodeSet which specifies a set of shared compartment configuration parameters.

Size: 1

class nxsdk.arch.n2a.graph.nodesets.dendrite_shared_cfg_obj_inf.DendriteSharedCfgObjInfWrapper(dendriteSharedCfgObjInf, parent, entryId=None)¶

Represents DendriteSharedCfg NodeSetInterface.

configure(**kwargs)¶

Allows to configure multiple register fields.

property disableInhibited¶

(uint1) Enables inhibition of compartment if voltage reaches negative limit defined by \(-2^negVmLimit\). The compartment is only able to leave the INHIBITED phase if it receives another excitatory input or after the refractory period if refractInhibited=1.

  • 0: Enable inhibition of compartment.

  • 1: Disable inhibition of compartment.

property dmOffsets¶

(uint1) Contributes to effective decay constant of compartment voltage via \(decayV + dmOffset\). Note that there are up to 32 different CxProfileCfg.decayV values but DendriteSharedCfg.dmOffset is shared by all compartments on a core.

property dsOffset¶

(uint1) Contributes to effective decay constant of compartment current via \(decayU + dsOffset\). Note that there are up to 32 different CxProfileCfg.decayU values but DendriteSharedCfg.dsOffset is shared by all compartments on a core.

fetch()¶
property gaussian¶
property negVmLimit¶

(uint5) Specifies exponent that defines negative limit of compartment voltage \(-2^negVmLimit\). Values > 23 are illegal.

property noiseAtDendOrVm¶

(uint1) Specifies whether either the compartment current or voltage gets randomized via noiseMantOffset0 and noiseExp0.

  • 0: Randomize current.

  • 1: Randomize voltage.

property noiseExp0¶

(uint6) Specifies the scaling exponent of the random number applied to the compartment current or voltage according to: \((LFSR() − 2^{7} + noiseMantOffset0 * 2^{6}) * 2^{noiseExp0 − 7}\) where \(LFSR()\) is are the bottom 8 bits of the DendriteRandom.word register.

property noiseExp1¶

(uint6) Specifies the scaling exponent of the random number applied to the compartment refractory period according to: \((LFSR() − 2^{7} + noiseMantOffset1 * 2^{6}) * 2^{noiseExp1 − 7}\) where \(LFSR()\) is are the bottom 8 bits of the DendriteRandom.word register.

property noiseMantOffset0¶

(int3) Specifies the mantissa offset of the random number applied to the compartment compartment current or voltage according to: \((LFSR() − 2^{7} + noiseMantOffset0 * 2^{6}) * 2^{noiseExp0 − 7}\) where \(LFSR()\) is are the bottom 8 bits of the DendriteRandom.word register.

property noiseMantOffset1¶

(int3) Specifies the mantissa offset of the random number applied to the compartment refractory delay according to: \((LFSR() − 2^{7} + noiseMantOffset1 * 2^{6}) * 2^{noiseExp1 − 7}\) where \(LFSR()\) is are the bottom 8 bits of the DendriteRandom.word register.

property posVmLimit¶

(uint3) Specifies exponent that defines positive limit of compartment voltage \(2^(posVmLimit*2+9)\).

property refractInhibited¶

(uint1) Allows compartment to leave INHIBITED phase after refractory period if disableInhibited=1 and compartment voltage saturated at \(-2^negVmLimit\).

  • 0: Compartment cannot leave INHIBITED phase.

  • 1: Comaprtment can leave INHIBITED phase after refractory period.
